SETTING-UP

1. Turn on the Wamp Server.
2. Set-up the node.js with npr.
3. Start the Netbeans 8.2 IDE with Tomcat Server.

CONFIGURATION
1. For the admin module, route to /log in.js.
2. Set the server's IP Address.

RUNNING EACH MODULE
1. For node.js, use the npm start on the root folder
2. For the Java, click the Run Project button in Netbeans or press F6. 
3. For the Php, just copy the souce codes and paste it into the Wamp Server.
